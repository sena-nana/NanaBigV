# NaNaBigV 文档

NaNaBigV 是一个面向主播的本地直播辅助产品，定位为 AI 观众氛围与直播控场助手，而不是单纯聊天机器人。

V1 固定为本地单机、单直播间、单操作者。系统以主播语音为主输入，Echo-Live 与视觉上下文只作为增强源；Provider 边界为 OpenAI 兼容 HTTP 接口；互动输出固定为 `danmaku`、`gift`、`super_chat`、`membership`。

## MVP 闭环

- 工作台：快速开始、继续上次配置、进入演练/中控台、连接状态、当前模型、预计消耗和最近复盘摘要。
- 新建直播：用 4 步向导完成直播类型、直播信息、AI 观众氛围和启动前检查。
- 直播中控台：围绕主播语音、Echo-Live 状态、候选弹幕、主播提词和控场按钮组织运行态。
- AI 观众：以观众组管理替代单个观众画像编辑。
- 话题库：管理今日大纲、话题卡和梗库，并标记使用阶段。
- 安全设置：管理输出模式、基础安全、质量过滤和频率上限。
- 弹幕记录：从本地队列与生成记录派生可筛选列表。

## 本地存储

NaNaBigV 通过 Tauri 本地 store 保存直播方案、观众组、话题库、安全设置和生成记录。默认输出模式为 `manual_review`，生成记录最多保留 200 条。
